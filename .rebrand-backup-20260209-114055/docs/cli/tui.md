# `autolab tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
autolab tui
autolab tui --url ws://127.0.0.1:18789 --token <token>
autolab tui --session main --deliver
```
